Vikunja lets you export all of your data and import tasks from other services, making it easy to back up your work or migrate from another tool.

## Export

![Data export section in settings](../../assets/images/help/import-export-data-export.png)

You can export all your data from Vikunja, including [projects](/help/projects), [tasks](/help/tasks), attachments, and more.

To start an export:

1. Open your **[Settings](/help/settings)** and navigate to the **Data Export** section.
2. Confirm your password to proceed.

<Callout type="info">
If you are using third-party authentication (for example, through Vikunja Cloud), you will see a confirmation step instead of a password prompt.
</Callout>

Once confirmed, Vikunja will generate a ZIP file containing all of your data. You can download this file and keep it as a backup or use it to re-import your data into Vikunja later.

## Import

![Import tab in settings](../../assets/images/help/import-export-import-tab.png)

To import data into Vikunja:

1. Open your **[Settings](/help/settings)** and navigate to the **Import** tab.
2. Select the source you want to import from.
3. Follow the on-screen steps to complete the import.

Vikunja supports importing from the following sources:

- **Vikunja export:** Re-import a previously exported ZIP file.
- **TickTick**
- **Todoist**
- **Trello**
- **Microsoft To Do**

You can also sync tasks with external apps using [CalDAV](/help/caldav).
